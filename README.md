# Helicopters_Project
Repositorio con los códigos para el proyecto de la asignatura de helicópteros. 

La organización del repositorio es la siguiente. 
BET: contiene el código/funciones para implementar el BET (blade element theory). 
MT: contiene el código/functiones para implementar el MT (momentum theory).
params.m: contiene los parámetros necesarios para BET y MT (tanto del helicóptero como del aire)
main.m: el main script en el que se ejecuta el código.
